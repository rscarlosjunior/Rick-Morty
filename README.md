# Rick and Morty Wikipedia Project

made with:

```
- vue
- vuex
- vue-router
- axios
- sass/scss
- jest
```

Run the project with:
```
npm install
npm run serve
npm run build
```
Run the tests with: 
```
npm run test:unit
```

API DOCUMENTATION : https://rickandmortyapi.com/
